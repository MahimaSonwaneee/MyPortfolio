# Portfolio Website

A **personal portfolio website** built using **HTML** and **CSS**, featuring a fully functional contact form integrated with **Formspree**.

## Features

- **Responsive Design** – Works on desktop and mobile devices.
- **About Me Section** – Showcase your skills and experience.
- **Portfolio/Projects Section** – Display professional projects.
- **Contact Form** – Users can send messages directly via **Formspree**.
- **Clean and Modern Layout** – Simple yet professional design using only HTML and CSS.

## Technologies Used

- **HTML5**
- **CSS3**
- **Formspree** (for contact form handling)
